Instructions
=================================
1. Download https://www.python.org/ftp/python/3.6.2/python-3.6.2-amd64.exe
2. Open igcla.py in a text editor and change the password on line 8. Save it.
2. Shift and right click on this folder, Select "Open command window here"
3. Type the following:
--------------------------------
python igcla.py

4. Press enter and wait
5. Open the file igcla_registration.csv
